# KARMA FX-T - Professional Trading Platform

A complete, feature-rich trading platform with live market data, authentication, and real-time analytics.

## Quick Start

```bash
npm install
npm start
```

Server runs on http://localhost:3000

## Features

- Live forex, crypto, and commodity data
- User authentication with JWT
- Real-time portfolio tracking
- Trading bot interface
- Community chat
- Professional dashboard
- Responsive design

## Project Structure

```
.
```
